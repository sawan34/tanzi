Manage bulk woocommerce product variations and attribute options.

== Description ==
NOTE: You must install the main "WooCommerce - excelling eCommerce" plugin in order to use this plugin

Manage bulk woocommerce product variations and attribute options.  You will be able to choose from a list of product categories, and then choose which attributes to assign to the products.

Plugin does not currently support price updates.  If you want to add unique variations prices you must do this by editing the individual products after bulk assigning the attributes.

This plugin also allows you to import a bulk list of attributes into the Woocommerce attribute admin.

== Installation ==

1. Unzip/Extract the plugin zip to your computer
2. Upload `\woocommerce-bulk-insert-attributes\` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

= To insert a bulk text list of attributes =
1. Go to WP Admin -> Products -> Attributes -> Add an attribute (such as "Color") -> Configure Terms
2. Paste your list of options into the Bulk Insert field; with each option on a new line

= To assign attributes to products =
1. Go to WP Admin -> Products -> Bulk Attribute Manager
